# Семейство методов Сагралова для изоляции вещественных и комплексных корней

## Файлы

### Основные реализации
- **Sagralov_ANewDsc.h** — метод ANewDsc для изоляции вещественных корней с использованием дискриминантных последовательностей
- **Sagralov_real_roots.h** — классический метод Сагралова для изоляции вещественных корней (VCA — Vincent–Collins–Akritas)
- **Sagralov_complex_roots.h** — метод CIsolate для изоляции комплексных корней в круговые диски

### Тестовые файлы
- **ANewDsc_real_roots_test.cpp** — тестирование метода ANewDsc
- **sagralov_real_roots_test.cpp** — тестирование классического метода вещественной изоляции
- **sagralov_complex_roots_test.cpp** — тестирование метода комплексной изоляции

## Назначение

Семейство методов Сагралова представляет собой алгоритмы изоляции корней алгебраических многочленов с действительными коэффициентами. Методы Сагралова находят непересекающиеся интервалы (для вещественных корней) или круговые диски (для комплексных корней), каждый из которых содержит ровно один корень многочлена.

Программа позволяет:
1. изолировать все вещественные корни многочлена в непересекающиеся интервалы;
2. изолировать все комплексные корни в круговые диски;
3. определять кратности корней;
4. работать с коэффициентами различных числовых типов (double, long double, float_precision).

---

## Алгоритмические особенности метода ANewDsc (Sagralov_ANewDsc.h)

1. **Дискриминантные последовательности**  
   Метод основан на построении последовательности Штурма для изоляции вещественных корней с использованием модифицированного алгоритма декартовой системы знаков.

2. **Адаптивное разбиение**  
   Интервалы поиска динамически разбиваются на подынтервалы до тех пор, пока каждый не будет содержать не более одного корня.

3. **Критерий завершения**  
   Изоляция считается завершённой, когда для каждого корня найден интервал, содержащий ровно один корень, и ширина интервала меньше заданного порога.

4. **Особенности**  
   - Эффективен для многочленов с хорошо разделёнными корнями
   - Чувствителен к наличию кратных корней
   - Время работы растёт с увеличением степени многочлена

---

## Алгоритмические особенности классического метода Сагралова (Sagralov_real_roots.h)

1. **Метод Vincent–Collins–Akritas (VCA)**  
   Классическая реализация алгоритма изоляции вещественных корней на основе теоремы Винсента и непрерывных дробей.

2. **Преобразование переменной**  
   Используются преобразования вида \(x \to \frac{1}{x+1}\) для локализации корней на положительной вещественной полуоси.

3. **Декартово правило знаков**  
   Количество положительных корней оценивается по числу перемен знаков в последовательности коэффициентов преобразованного многочлена.

4. **Преимущества**  
   - Высокая точность изоляции
   - Устойчивость к кратным корням (изоляция кратных корней возможна, но интервалы могут сливаться)
   - Хорошая масштабируемость на многочлены высокой степени

---

## Алгоритмические особенности метода комплексной изоляции (Sagralov_complex_roots.h)

1. **Изоляция в круговые диски**  
   Метод CIsolate находит для каждого комплексного корня круговой диск, не содержащий других корней.

2. **Алгоритм**  
   - Вычисление границ корней по теореме Коши
   - Разбиение комплексной плоскости на прямоугольные области
   - Применение принципа аргумента для подсчёта корней в области
   - Рекурсивное уточнение до изоляции каждого корня

3. **Особенности обработки**  
   - Вещественные корни изолируются на вещественной оси как интервалы
   - Комплексные корни изолируются как круговые диски
   - Сопряжённые комплексные корни обрабатываются парами

## Анализ результатов

### Успешные сценарии

| Метод | Успешные сценарии |
|-------|-------------------|
| **ANewDsc** | Вещественные корни до степени 30 (double), до 50 (long double), до 70 (float_precision); кластеризованные корни до степени 40 при использовании float_precision |
| **CIsolate** | Комплексные корни до степени 70 (float_precision); вещественные корни степени 5 и 15; высокая степень (25–30) |
| **Классический Сагралов** | Вещественные корни до степени 40 (double), до 50 (long double), до 80 (float_precision); кластеризованные корни до степени 28; высокая степень до 80 |

### Проблемные сценарии

| Проблема | Проявление | Затронутые методы |
|----------|------------|-------------------|
| **Кратные корни** | Сильное снижение качества изоляции, интервалы сливаются | ANewDsc, CIsolate, классический Сагралов (float_precision) |
| **Степень > 40–50** | Резкое падение процента покрытых корней | ANewDsc (все типы) |
| **Вещественные корни степени 10, 20** | Нулевое покрытие для некоторых тестов | CIsolate |
| **Кластеризация > 30** | Снижение точности изоляции | ANewDsc, классический Сагралов (float_precision) |

### Сравнительная характеристика

| Реализация | Преимущества | Ограничения |
|------------|--------------|-------------|
| **ANewDsc** | Эффективен для хорошо разделённых корней, работает с произвольной точностью | Чувствителен к кратным корням, теряет эффективность при степени > 50 |
| **CIsolate** | Изоляция комплексных корней, работает с произвольной точностью до степени 70 | Проблемы с вещественными корнями на некоторых степенях, чувствителен к кластеризации |
| **Классический Сагралов (VCA)** | Наиболее стабильный метод, успешно изолирует корни до степени 80, устойчив к кластеризации | Кратные корни требуют дополнительной обработки при использовании float_precision |

## Выводы

Семейство методов Сагралова демонстрирует высокую эффективность для изоляции вещественных и комплексных корней алгебраических многочленов.

**Ключевые результаты:**
- Классический метод Сагралова (VCA) показывает наилучшие результаты среди всех реализаций, успешно проходя 14 из 16 тестов для double, 16 из 16 для long double и 34 из 49 для float_precision
- Метод CIsolate эффективно изолирует комплексные корни, особенно при использовании арифметики произвольной точности (14 из 19 тестов)
- Метод ANewDsc демонстрирует приемлемые результаты для многочленов до степени 50, но значительно уступает классической реализации

**Рекомендации по выбору реализации:**
1. **Для изоляции вещественных корней многочленов степени до 80** — классический метод Сагралова (Sagralov_real_roots.h)
2. **Для изоляции комплексных корней многочленов степени до 70** — метод CIsolate с типом float_precision
3. **Для многочленов с хорошо разделёнными корнями и степенью до 50** — метод ANewDsc
4. **Для многочленов с кратными корнями** — требуется дополнительная постобработка или использование классического метода Сагралова с последующим уточнением
