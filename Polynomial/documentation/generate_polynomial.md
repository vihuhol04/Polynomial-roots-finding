# Модуль generate_high_degree_polynomial.h
Модуль реализует универсальный генератор многочленов высокой степени с 
контролируемой структурой корней. Основная функция — 
generate_high_degree_polynomial — позволяет формировать многочлены с заданными 
параметрами, включая:
	- количество пар комплексных сопряжённых корней;
	- количество кластеров вещественных корней;
	- радиусы кластеров и их распределение;
	- группы кратных корней с указанием их степеней;
	- параметр нормировки коэффициентов;
	- инициализационное значение случайного генератора (seed).
