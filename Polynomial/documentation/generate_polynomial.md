# Модуль generate_high_degree_polynomial.h
Модуль реализует универсальный генератор многочленов высокой степени с 
контролируемой структурой корней. Основная функция — 
generate_high_degree_polynomial — позволяет формировать многочлены с заданными 
параметрами, включая:
	1. количество пар комплексных сопряжённых корней;
	2. количество кластеров вещественных корней;
	3. радиусы кластеров и их распределение;
	4. группы кратных корней с указанием их степеней;
	5. параметр нормировки коэффициентов;
	6. инициализационное значение случайного генератора (seed).
