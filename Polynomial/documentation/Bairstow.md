# Метод Бэрстоу (Bairstow) для вычисления корней многочленов

## Файл
- **Bairstow.h** — реализация метода Бэрстоу для нахождения всех корней алгебраических многочленов с действительными коэффициентами.

## Назначение
Метод Бэрстоу представляет собой итерационный алгоритм для последовательного выделения квадратичных множителей многочлена, что позволяет находить как вещественные, так и комплексно-сопряжённые корни. В отличие от методов, работающих с модулями корней, метод Бэрстоу непосредственно находит корни с высокой точностью.

Программа позволяет:
1. находить все корни многочлена (вещественные и комплексные);
2. определять кратные корни;
3. работать с коэффициентами различных числовых типов (float, double, long double и произвольной точности).

---

## Алгоритмические особенности метода Бэрстоу

1. **Выделение квадратичных множителей**  
   Метод основан на итерационном поиске коэффициентов квадратичного множителя \(x^2 + px + q\), который является делителем исходного многочлена. Это позволяет одновременно находить пару комплексно-сопряжённых корней или два вещественных корня.

2. **Метод Ньютона для двух переменных**  
   Поиск коэффициентов \(p\) и \(q\) осуществляется с использованием обобщения метода Ньютона для системы двух нелинейных уравнений, где функции — это остатки от деления многочлена на квадратичный множитель.

3. **Дефляция (понижение степени)**  
   После нахождения квадратичного множителя исходный многочлен делится на этот множитель, что позволяет последовательно обрабатывать все корни. Для вещественных корней возможно выделение линейных множителей.

4. **Критерий завершения**  
   Итерации прекращаются, когда изменение коэффициентов \(p\) и \(q\) становится меньше заданного порога `epsilon` или достигнуто максимальное число итераций.

5. **Обработка особых случаев**  
   - Константный многочлен — результат пустой.
   - Все коэффициенты равны нулю — результат пустой.
   - Вещественные корни обрабатываются как частный случай квадратичного множителя с нулевой мнимой частью.
   - Кратные корни определяются по повторяющимся множителям.

6. **Численная устойчивость**  
   - Использование адаптивных порогов сходимости.
   - Контроль переполнения при больших степенях.
   - Применение масштабирования коэффициентов при необходимости.

2. **100% успешность на арифметике произвольной точности (float_precision)** — метод успешно прошёл все 11 тестов, включая многочлены степени 100, что подтверждает его устойчивость при использовании повышенной разрядности.

## Особенности реализации

1. **Последовательная дефляция**  
   Метод последовательно выделяет квадратичные множители, что позволяет обрабатывать многочлены любой степени, но приводит к накоплению ошибок при большом числе шагов.

2. **Обработка вещественных корней**  
   При обнаружении вещественного корня (\(q \approx 0\)) метод переключается на выделение линейного множителя, что повышает эффективность для многочленов с вещественными корнями.

3. **Контроль сходимости**  
   Используются адаптивные пороги, учитывающие масштаб коэффициентов и текущую степень многочлена.

4. **Защита от зацикливания**  
   Реализовано ограничение на максимальное число итераций для каждого квадратичного множителя.
