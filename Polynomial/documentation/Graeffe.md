# Назначение
Разработанный программный модуль предназначен для вычисления модулей корней 
алгебраического многочлена с действительными коэффициентами. Основой метода 
является алгоритм Грефе, реализованный в устойчивой логарифмической форме.

Программа позволяет:
	1) находить модули корней многочлена;
	2) определять кратности корней по совпадающим модулям;
	3) работать с коэффициентами различных числовых типов 
	(float, double, long double и произвольной точности).

# Алгоритмические особенности метода Грефе для вещественных корней
	1. Использование логарифмов
	Все операции выполняются над логарифмами коэффициентов для предотвращения 
	переполнения при многократном возведении в квадрат.
	2. Функция безопасного логарифма
	Для обработки нулевых коэффициентов используется safe_log(x), возвращающая 
	-inf при x = 0.
	3. Критерий завершения
	Итерации прекращаются, когда относительное изменение коэффициентов меньше
	epsilon или достигнут maxIter.
	4. Обработка особох случаев
		4.1 Константный многочлен — результат пустой.
		4.2 Все коэффициенты равны нулю — результат пустой.
		4.3 Наличие промежуточных нулевых коэффициентов обрабатывается без ошибок

# Результаты тестирования
Тестирование проведено на пяти группах полиномов различной степени и структуры.
Все тесты выполнены с параметрами epsilon = 1e-10, maxIter = 100.
Реализация метода Грефе показала устойчивую и предсказуемую работу при вычислении 
модулей корней многочленов различной степени и структуры. Алгоритм корректно 
определяет модули корней для большинства тестовых полиномов и демонстрирует 
высокую численную устойчивость благодаря использованию логарифмической формы 
вычислений.
При полиномах малой и средней степени метод обеспечивает высокую точность и 
быструю сходимость. Для полиномов большой степени (более 20) результаты сохраняют 
корректность, хотя наблюдается частичное сглаживание различий между малыми 
модулями. В случаях наличия кратных или сильно сближенных корней точность 
определения отдельных модулей снижается, что соответствует известным ограничениям 
классического метода Грефе.


# Алгоритмические особенности метода Грефе для вещественных корней
	1. Обработка различных случаев
	Вещественные корни, комплексные корни, кратные корни, нулевые корни
	2. Численная устойчивость
	Использование логарифмической арифметики, адаптивные пороги для разных
	масштабов коэффициентов, множественная валидация кандидатов
	3. Оптимизации
	Предвычисление производных полинома, кэширование промежуточных
	результатов, дедупликация найденных корней
	4. Работа на основе метода Грефе
	В Graeffe_real_roots.h вычисляются модули ВСЕХ корней (в т.ч. комплексных),
	и на их основе находятся действительная и мнимая части комплексного числа

# Результаты тестирования
Модифицированный метод Хосмана в целом подтвердил свою работоспособность 
и устойчивость в стандартных случаях, однако не прошёл все тесты с 
одинаковым качеством.
Метод обеспечивает корректные результаты для простых и умеренно сложных 
многочленов, но демонстрирует снижение точности при высокой степени и 
кратных корнях. Дальнейшее развитие требует доработки алгоритмов 
нормализации коэффициентов и уточняющих процедур для повышения точности 
определения комплексных и близких по модулю корней.
