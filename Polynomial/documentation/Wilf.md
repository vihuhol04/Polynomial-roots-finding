# Метод Вилфа (Wilf) для вычисления корней многочленов

## Файл
- **Wilf.h** — реализация метода Вилфа для нахождения всех корней алгебраических многочленов с действительными коэффициентами.

## Назначение
Метод Вилфа представляет собой итерационный алгоритм для одновременного нахождения всех корней многочлена. Метод Вилфа позволяет непосредственно находить комплексные корни с высокой точностью.

Программа позволяет:
1. находить все корни многочлена (вещественные и комплексные);
2. определять кратные корни;
3. работать с коэффициентами различных числовых типов (float, double, long double и произвольной точности).

---

## Алгоритмические особенности метода Вилфа

1. **Одновременная итерация**  
   Метод основан на одновременном уточнении всех корней многочлена с использованием симметрических функций от корней.

2. **Преобразование коэффициентов**  
   На каждом шаге вычисляются новые приближения корней через преобразования, аналогичные методу Бернулли, но применяемые ко всем корням одновременно.

3. **Сходимость**  
   Метод демонстрирует квадратичную сходимость для простых корней и линейную для кратных.

4. **Численная устойчивость**  
   Используются адаптивные пороги и защита от переполнения при работе с коэффициентами.

5. **Обработка особых случаев**  
   - Константный многочлен — результат пустой.
   - Нулевые корни обрабатываются корректно.
   - Кратные корни определяются по сходимости приближений.

### Проблемные сценарии

| Проблема | Проявление | Затронутые типы |
|----------|------------|-----------------|
| **Высокая степень (>28) для комплексных корней** | Резкое падение процента найденных корней при степени ≥30 | float_precision |
| **Кластеризация высокой степени (>16)** | Снижение эффективности при степени ≥18 | float_precision |
| **Масштабируемость** | При степени 44 метод находит лишь 28 из 44 корней (64%) | float_precision |

| Тип данных | Типичная максимальная невязка \|P(z)\| |
|------------|---------------------------------------|
| **double** | 10⁻¹¹ – 10⁻¹² |
| **long double** | 10⁻¹¹ – 10⁻¹² |
| **float_precision** | 10⁻⁷ – 10⁻⁴ |

### Ключевые результаты

1. **Метод Вилфа демонстрирует наивысшую точность среди всех рассмотренных методов** для типов double и long double, достигая невязки порядка 10⁻¹¹ – 10⁻¹² на всех пройденных тестах.

2. **100% успешность на double и long double** — метод прошёл все 13 тестов без единого сбоя, включая тесты с вещественными, комплексными, кластеризованными, кратными корнями и многочленами степени до 20.

3. **Ограничение по степени** — на арифметике произвольной точности (float_precision) метод успешно работает до степени 28 для комплексных корней и до степени 16 для кластеризованных, после чего эффективность резко снижается.

4. **Высокая точность на float_precision** — даже при частичной сходимости метод обеспечивает невязку порядка 10⁻⁴ – 10⁻⁷, что является приемлемым для многих практических задач.
